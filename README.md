## ✨ Features / বৈশিষ্ট্যসমূহ 

### 🇬🇧 English
- **Sticky Header:** The `Live Sport` logo and `● ON AIR` status remain locked at the top while scrolling.
- **Single Page Scroll:** Smooth scrolling transition from the channel list back to the player.
- **8-Channel Layout:** Clean, fully responsive card layout for up to 6 distinct servers/channels.
- **CORS Bypass Ready:** Integrates a proxy backend to prevent browser media loading errors.

### 🇧🇩 বাংলা
- **স্টিকি হেডার:** স্ক্রল করার সময়ও `Live Sport` লোগো এবং `● ON AIR` নোটিফিকেশনটি স্ক্রিনের ওপরে ফিক্সড থাকবে।
- **সিঙ্গেল পেজ স্ক্রলিং:** নিচের চ্যানেলে ক্লিক করলেই পেজটি ম্যাজিকের মতো স্মুথলি স্ক্রল হয়ে উপরে প্লেয়ারে চলে যাবে।
- **৮টি চ্যানেল লেআউট:** স্কেচ অনুযায়ী সাজানো ৬টি আলাদা সার্ভার বা চ্যানেল বক্স, যা মোবাইল ও পিসিতে অটো-ফিট হয়।
- **CORS বাইপাস:** ব্রাউজারে ভিডিও লোডিং এরর বা ব্লক হওয়া আটকাতে প্রক্সি সেটআপ যুক্ত।
